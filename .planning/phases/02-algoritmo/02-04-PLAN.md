  <name>Task 1: Server Actions — triggerGeneration, triggerGenerationDay, triggerGenerationRange</name>
  <files>lib/actions/generation.ts</files>
  <action>
    Create `lib/actions/generation.ts` with `'use server'` at top. Three exported async functions:

    **triggerGeneration(month: number, year: number): Promise&lt;{ success: boolean; logCount: number; errors: number }&gt;**
    - Auth check: get session via `auth.api.getSession({ headers: await headers() })`, throw if missing
    - Load data in parallel: `Promise.all([getEmployeesWithAreas(), getShiftTemplates(), getAbsencesForMonth(month, year)])`
    - Call `generateMonthShifts({ month, year, employees, templates, absences })`
    - Save: `await saveGeneratedShifts(result.shifts)` then `await saveGenerationLog(month, year, result.log)`
    - Revalidate: `revalidatePath('/planilla')` and `revalidatePath('/bitacora')`
    - Return summary: `{ success: true, logCount: result.log.length, errors: result.log.filter(l => l.type === 'error').length }`
    - On any error: catch, log error entry, return `{ success: false, logCount: 0, errors: 1 }`

    **triggerGenerationDay(date: string): Promise&lt;{ success: boolean }&gt;**
    - Parse date string to get month/year
    - Same pattern as triggerGeneration but only for a single day
    - Filter templates and employees for that specific date before calling the service
    - Implementation note: call generateMonthShifts with the full month data, then filter result.shifts to only the target date before saving — simplest correct approach

    **triggerGenerationRange(startDate: string, endDate: string): Promise&lt;{ success: boolean; daysGenerated: number }&gt;**
    - Parse start/end dates
    - Call triggerGeneration for each month that the range overlaps
    - For partial months, filter output to the date range before saving
    - Return count of days generated

    All three functions use `'use server'` (already declared at file top). Auth check in each.

    Imports needed:
    - `'use server'` at file top
    - `import { auth } from '@/lib/auth'`
    - `import { headers } from 'next/headers'`
    - `import { revalidatePath } from 'next/cache'`
    - `import { getEmployeesWithAreas } from '@/lib/db/employees'`
    - `import { getShiftTemplates } from '@/lib/db/templates'`
    - `import { getAbsencesForMonth } from '@/lib/db/absences'`
    - `import { saveGeneratedShifts } from '@/lib/db/shifts'`
    - `import { saveGenerationLog } from '@/lib/db/generationLog'`
    - `import { generateMonthShifts } from '@/lib/services/generation'`
