  <behavior>
    - GET /api/export?year=2025&month=1 returns 200 with Content-Type application/vnd.openxmlformats-officedocument.spreadsheetml.sheet and Content-Disposition attachment; filename="planilla-enero-2025.xlsx"
    - GET /api/export with missing year or month returns 400 with JSON { error: "year and month are required" }
    - GET /api/export?year=abc&month=1 returns 400 with JSON { error: "year and month must be integers" }
    - The .xlsx file header row starts with "ASESOR" in column A, then day headers LUN 01, MAR 02... (UPPERCASE, Spanish day abbreviation + zero-padded day number) for every day in the month
    - timeSlot null -> cell is blank
    - timeSlot "DESCANSO" or shift with no timeSlot and no areaId -> cell text "DESCANSO"
    - timeSlot "7:00-14:00" -> cell text "7:00-14:00" (single line)
    - timeSlot "7:00-11:00|11:30-14:30" -> cell text "7:00-11:00\n11:30-14:30" (two lines, wrapText: true on cell alignment)
  </behavior>
  <action>
    Create `app/api/export/route.ts` as a Next.js Route Handler (NOT a Server Action).

    Architecture pattern (from research/STACK.md):
    - Route Handler returns `Response` (not NextResponse) with the binary buffer
    - ExcelJS writes to buffer in memory via `workbook.xlsx.writeBuffer()`
    - Do NOT use Server Actions for file downloads — they are POST-only and cannot stream binary

    Implementation steps:

    1. Parse query params: `year` and `month` from `request.nextUrl.searchParams`. Validate both are present and parse as integers. Return 400 JSON on error.

    2. Compute the date range: all days from `new Date(year, month - 1, 1)` to `new Date(year, month - 1, lastDay)`. Use `getDaysInMonth` from date-fns or compute with `new Date(year, month, 0).getDate()`.

    3. Query all shifts for the month with employee names:
       ```typescript
       const shifts = await db.shift.findMany({
         where: {
           date: {
             gte: new Date(year, month - 1, 1),
             lte: new Date(year, month - 1, daysInMonth),
           },
         },
         include: { employee: true },
         orderBy: [{ employee: { name: "asc" } }, { date: "asc" }],
       });
       ```

    4. Build a lookup map: `Map<employeeId, Map<dateString, Shift>>` for O(1) cell access.

    5. Get distinct employees ordered by name from the shifts (or query all employees separately and include only those with at least one shift or include all employees).
       - Include ALL employees (query `db.employee.findMany({ orderBy: { name: "asc" } })`) so the spreadsheet matches the full planilla even for employees with no shifts that month.

    6. Build ExcelJS workbook:
       ```typescript
       import ExcelJS from "exceljs";
       const workbook = new ExcelJS.Workbook();
       const sheet = workbook.addWorksheet("Planilla");
       ```

    7. Day abbreviations in UPPERCASE Spanish. Use a hardcoded map (do NOT rely on date-fns locale for UPPERCASE output — use toUpperCase() explicitly):
       ```typescript
       const DAY_ABBR = ["DOM", "LUN", "MAR", "MIE", "JUE", "VIE", "SAB"];
       // index by getDay() (0=Sunday, 1=Monday... 6=Saturday)
       ```

    8. Header row: cell A1 = "ASESOR" (bold), then one column per day: `DAY_ABBR[dayOfWeek] + " " + String(dayNum).padStart(2, "0")` (e.g. "LUN 01", "MAR 02"). Bold header row, set row height.

    9. Employee rows: one row per employee. Column A = employee.name. For each day column:
       - Find shift in lookup map using date key `yyyy-MM-dd` format
       - If no shift record exists: leave cell blank
       - If shift.timeSlot is null: write "DESCANSO"
       - If shift.timeSlot contains "|": split on `|`, join with `\n`, set `cell.alignment = { wrapText: true, vertical: "top" }`
       - Otherwise: write timeSlot as-is

    10. Set column widths: column A = 22 (employee names), all day columns = 12.

    11. Freeze first row and first column (sticky headers):
        ```typescript
        sheet.views = [{ state: "frozen", xSplit: 1, ySplit: 1 }];
        ```

    12. Write buffer and return:
        ```typescript
        const buffer = await workbook.xlsx.writeBuffer();
        const monthName = format(new Date(year, month - 1, 1), "MMMM-yyyy", { locale: es });
        return new Response(buffer as ArrayBuffer, {
          headers: {
            "Content-Type": "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
            "Content-Disposition": `attachment; filename="planilla-${monthName}.xlsx"`,
          },
        });
        ```

    Import `db` from `@/lib/db` (Prisma singleton, not `new PrismaClient()`).
    Import `format` from `date-fns` and `es` from `date-fns/locale/es` for month name in filename.
    ExcelJS must only be imported server-side — this file is a Route Handler (server-only), so no "use client" directive needed.
